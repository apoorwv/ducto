# JavaScript API

ducto for TypeScript/JavaScript is published as [`@apoorwv/ducto`](https://www.npmjs.com/package/@apoorwv/ducto) on npm. Requires Node.js 18+.

## Installation

```bash
npm install @apoorwv/ducto

# PostgreSQL store (optional)
npm install pg

# YAML pricing loading (optional)
npm install js-yaml
```

## Core Classes

| Class / Type | Exported from | Description |
|---|---|---|
| [`PricingEngine`](/docs/javascript-api/pricing-engine) | `@apoorwv/ducto` | Pure calculation layer — no DB dependency |
| [`CreditManager`](/docs/javascript-api/credit-manager) | `@apoorwv/ducto` | Orchestrates credit lifecycle |
| [`HttpxSupabaseStore`](/docs/javascript-api/stores) | `@apoorwv/ducto` | Supabase-backed store (native `fetch`, no extra deps) |
| [`PostgresStore`](/docs/javascript-api/stores) | `@apoorwv/ducto` | Direct PostgreSQL via `pg` |
| [`MemoryStore`](/docs/javascript-api/stores) | `@apoorwv/ducto/node` | In-memory store for testing (Node.js only) |
| `CreditEventEmitter` | `@apoorwv/ducto` | Typed pub/sub for credit lifecycle events |
| `UsageMetrics` | `@apoorwv/ducto` | Input interface for `calculate()` / `deduct()` |
| `CostBreakdown` | `@apoorwv/ducto` | Structured cost breakdown (all fields `Decimal`) |
| `CreditManagerOptions` | `@apoorwv/ducto` | Constructor options (policy, thresholds, TTL, …) |
| `PricingConfigData` | `@apoorwv/ducto` | Schema for a pricing config dict |
| `DeductionResult` | `@apoorwv/ducto` | Return type of `deduct()` / `settle()` |
| `LeaseResult` | `@apoorwv/ducto` | Return type of `reserve()` / `renew()` |
| `BalanceResult` | `@apoorwv/ducto` | Return type of `getBalance()` |
| `CanAffordResult` | `@apoorwv/ducto` | Return type of `canAfford()` |

All money values (balances, amounts, costs) are `Decimal` from `decimal.js`, never binary `number`.

## Quick Example

```typescript
import { CreditManager, PostgresStore } from "@apoorwv/ducto";
// For tests / local dev, use MemoryStore from the Node subpath:
import { MemoryStore } from "@apoorwv/ducto/node";

const store = new MemoryStore();
const manager = new CreditManager(store);

await manager.publishPricingFromDict({
  models: { "gpt-4": "input_tokens * (0.01 / 1000) + output_tokens * (0.03 / 1000)" },
});

await manager.addCredits("user_abc", 1000);
const result = await manager.deduct(
  "user_abc",
  { model: "gpt-4", inputTokens: 500, outputTokens: 200 },
  "idempotency-key-123",
);
// result.amount is a Decimal — use .toString() or .toNumber()
console.log(`Deducted ${result.amount.toString()} credits`);
```
